# Tutorials

Most people that learn Roux have a prior cubing background. However, if you've never solved a Rubik's Cube before, some people still learn Roux as their first ever method.

{{% details "I already know how to solve a Rubik's Cube. How do I learn Roux?" %}}
The best way to get started is to use this [Roux guide](https://www.youtube.com/watch?v=ESYJ-DFx0Qo&list=PL754ADDA2E38A21AA).

Alternatively, you can use [Kian's tutorial](https://www.youtube.com/playlist?list=PLBHocHmPzgIjnAbNLHDycgaCP5IqiwnU9) which is aimed for complete beginners, but is still fairly informative.
{{% /details %}}

{{% details "I don't know how to solve a Rubik's Cube. How can I get started?" %}}
Despite Roux being a speedsolving method, it is also a fairly good beginner method.

[Kian's tutorial](https://www.youtube.com/playlist?list=PLBHocHmPzgIjnAbNLHDycgaCP5IqiwnU9) is pretty good, but there is also a [web-based Roux tutorial that you can use instead](https://tutorial.rouxers.com/).
{{% /details %}}
